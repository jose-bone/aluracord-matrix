## 🚀 Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/jose-bone/aluracord-matrix.git && cd aluracord-matrix
```

Follow the steps below

```bash
# Install the dependencies
$ yarn install

# Start the project
$ yarn dev
```
